여러가지 형태의 데이터(속성)들을 한 곳에 묶어서 하나의 그룹으로 관리를 쉽게 하겠다! > 구조화
구조화를 위해 class를 사용!(나중에 class는 캡슐화에도 사용)
또한, 어떤 클래스를 구조화하는 더 큰 범위도 만들 수 있다.

class는 참조형으로 선언해서 사용 가능하다(new연산자로 공간을 만들어야 한다!)
Member[] list = new Member[10]; member 참조변수가 들어갈 방!!(일반 값이 들어가는 공간이 아니다)
그래서 각 방마다 객체를 생성해줘야한다.
list[0] = new Member();  >> 참조해서 사용할 객체를 생성 해 주어야 함.

Member : 개체(개념상의 그룹핑명)
list : 객체(개체를 실체화 한 것,)

힙과 스택.
메인메모리를 반으로 나누어 데이터영역과 코드 영역으로 사용한다.
프로그램을 실행시키면, 프로그램 코드가 코드 영역으로 할당된다.(이때 상태는 프로세스라고 한다)
데이터영역은 힙과 스택 영역으로 구분된다.
입석자리와 예약자리처럼 서로 다른 곳으로부터 채워지기 시작한다.
예약자리는 스택영역이다.
main 함수가 실행될 때는 함수 내부의 모든 변수를 준비 시켜둔 상태에라야 실행이 가능하다.
이 변수들을 스택영역에 준비한다.main(String args[]) 매개변수 포함.
힙 영역에는 new 연산자로 생성한 공간이 마련이 된다.
