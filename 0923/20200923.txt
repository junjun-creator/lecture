코드 작성순서

1. 로또(데이터) - entity package
2. 로또 서비스(기능) - service package
3. 로또 컨트롤러(ui) - controller package

무조건 다 함수를 각각 구현해서 메인함수에서 호출하려고 하지 말자.
함수 안에서 다른 함수를 호출하여 여러 기능을 연결 해도 된다.

>>> 객체지향적으로 프로그래밍 하는 법을 생각해보자. 좀더 나은 방향으로

객체지향 프로그래밍이란?
Object(객체) Oriented(중심) Programming
움직이는 배경에 아군 전투기와 적군 전투기가 있다.
적군 전투기가 미사일을 쏘면 아군전투기는 마우스 조작으로 피하고,
미사일을 쏴서 적군 전투기를 맞추면 터진다.

이 경우에 객체는 배경, 아군전투기, 적군전투기, 미사일이 된다.
함수는 각 객체들의 행위를 가리킨다.


전체 겉의 구조는 우리가 일상생활 우리가 생각한 그대로 구조화 시키고, 
그 구조 안에서 기능을 담당하는 코드는 절차와 함수로 구현하면 된다.

우리가 말하는 것에서 명사가 객체(클래스) 이다.
동사는 함수이다.
