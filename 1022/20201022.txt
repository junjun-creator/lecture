컬렉션
1.컬렉션이란?
>가변길이 저장소
2. 열거서비스란?
> 인덱스를 사용하지 않고 값을 반복적으로 꺼내는 서비스 next()/ hasNext()
3.반복자(Iterator) 란?
> 스레드에 안전한 열거 서비스를 제공하기 위한 열거 객체
4. foreach 구문이란?
> 콜렉션을 순회해주는 반복문(콜렉션이 iterable 인터페이스를 구현하고 있어야만 사용 가능)
	
	Key
Set :	값	중복 제거(인덱스가 없어서 특정 데이터 꺼내기 불가능)
List :	인덱스	목록 구현용
Map:	명칭   객체를 대신할수 있는 임시객체

Tree >> 데이터를 자주 조회하는 경우
Hash >> 삽입 삭제가 자주 일어나는 경우
Concurrent >> 스레드가 많은 경우
