git

로컬저장소 : 사진찍어서(commit) 보관하는 곳
사진을 찍기 위해 인덱스에 추가해야함(add staging)
사진을 찍을때 누구와 찍엇는지 코멘트(commit -m "")

untracked > tracked > committed

환경변수
global, system, local

commit 취소 방법

[방법 1] 커밋을 지우고 파일 상태를 unstaged로 만든다.

- git reset --soft 

[방법 2] 커밋을 지우고 파일 상태를 staged로 만든다.

- git reset --mixed (기본 설정; git reset 을 하면 --mixed 옵션이 기본으로 적용됨)

[방법 3] 커밋을 지우고 파일의 변경사항도 지워버린다.

- git reset --hard
